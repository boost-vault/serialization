Summary of bitset_and_more

Author: Steven F. McKay, Steve.F.McKay@ssa.gov, April 11, 2006.

The files in this zip file apply to the Boost Library, version 1.33.1.

The problem

The serialization library does not have the ability to serialize a std::bitset. When serializing a std::vector<bool> (a close cousin to a std::bitset), it stores each bit in one byte (in binary mode). As a result, an archive can be much larger than necessary.

The solution

The files in this zip file add the capability to serialize a std::bitset, and the ability to save 8 bits in one byte (in binary mode) for a std::bitset and a std::vector<bool>. Note that there are only 3 files that actually implement the changes; the remaining files are documentation.
